Microsoft упростила процесс «чистой» установки Windows 10.
Компания Microsoft начала тестировать новый инструмент, значительно упрощающий «чистую» установку Windows 10. Ранее пользователям операционной системы предлагалась возможность полностью «перезагрузить» компьютер и вернуть его к оригинальному состоянию. Тестируемый же сейчас инструмент позволит стереть всё заранее установленное производителем компьютера программное обеспечение и установить полноценную «чистую» версию Windows 10. Этот вариант идеально подойдёт для новых ПК, которые пользователь может захотеть быстро и безболезненно очистить от ненужных программ.
Инструмент пока доступен лишь в тестовой версии Windows 10, однако, скорее всего, полноценный его релиз должен состояться довольно скоро. Выход юбилейного обновления Windows 10 должен состояться в следующем месяце — в новой версии, скорее всего, появится и инструмент для «чистой» установки операционной системы. Сейчас Microsoft активно ищет баги в своём продукте.
Также компания внесла ряд изменений в Windows 10 Mobile. В частности, из мобильной версии ОС убрали функцию «Детский уголок», поскольку ею почти никто не пользовался. «Использование «Детского уголка» было слишком низким, чтобы оправдать продолжение разработки и поддержки функции», — заявила Дона Саркар (Dona Sarkar), глава программы Windows Insider. Тем не менее, в Windows 10 Mobile по-прежнему есть «Уголок приложений», позволяющий создать отдельный ограниченный профиль с определёнными приложениями и уникальным домашним экраном.
