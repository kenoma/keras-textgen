Microsoft может перезапустить Windows Movie Maker для Windows 10.
Мы помним Movie Maker как довольно удачное решение — простой и функциональный видеоредактор, совместимый со старыми версиями Windows, однако в Windows 10 компания Microsoft отказалась от поддержки этого приложения, и пользователи вынуждены искать ему замену.
Тем не менее, появилась надежда, что Microsoft может когда-нибудь вернуть Windows Movie Maker в обновлённом виде. Представители компании намекнули, что этот проект разрабатывается в закрытом режиме.
По сообщениям WindowsCentral, обновление на странице поддержки сайта Microsoft гласит: «Movie Maker скоро будет доступен для пользователей Windows 10 в магазине приложений для Windows», так что, похоже, титан программного обеспечения действительно собирается выпустить универсальное приложение с той же функциональностью, совместимое с ПК и мобильными устройствами на базе Windows 10.
На данный момент какие-либо подробности отсутствуют: на русскоязычной версии сайта Microsoft нет даже раздела, посвящённого альтернативным приложениям, поэтому сложно с уверенностью утверждать, появится всё-таки поддерживаемая версия Movie Maker для Windows 10 или нет.
Так Windows Movie Maker выглядит сейчас
Безусловно, запуск современной версии приложения необходим, поскольку, помимо прочего, он поможет Microsoft поднять уровень универсальных приложений, однако представители компании окружают завесой тайны всё, что касается данного проекта. На странице официального сайта, посвящённой Windows Movie Maker, список систем, совместимых с приложением, ограничен Windows 8.1, Windows 8, Windows 7; также указано, что программа недоступна для Windows RT 8.1 и Windows RT. Windows 10 там даже не упоминается.
Пользователи Windows 10 по-прежнему могут загрузить и установить Movie Maker (для русской версии — «Фотоальбом и Киностудия»), однако приложение уже давно не обновлялось, и велика вероятность, что обновлений так и не появится вплоть до 2017 г., когда Microsoft планирует отказаться от всего пакета основных компонентов Windows от 2012 г.
Окончание срока поддержки назначено на 10 января 2017 г., поэтому у пользователей Windows 10 всё ещё есть шанс вновь увидеть Movie Maker, например, в рамках запуска Redstone 2 весной будущего года.
