Доля Windows 10 практически перестала расти.
Компания Net Applications обнародовала статистику по мировому рынку операционных систем для персональных компьютеров по итогам октября 2016 года.
Самой распространённой программной платформой для ПК продолжает оставаться Windows 7. Причём доля этой ОС за последний месяц даже незначительно выросла, несмотря на усилия Microsoft по продвижению флагманской системы Windows 10. Так, если в сентябре Windows 7 занимала около 48,3 % глобального рынка, то теперь — 48,4 %. В октябре 2015-го на Windows 7 приходилось 55,7 %.
Вместе с тем доля Windows 10, которая находится на втором месте по популярности, практически перестала расти. За прошедший месяц эта операционная система совсем незначительно упрочила позиции — с 22,53 % до 22,59 %. Возможно, это отчасти объясняется тем, что с конца июля Microsoft прекратила бесплатное обновление более ранних версий ОС до Windows 10.
На третье место вышла операционная система Windows 8.1 с результатом в 8,4 % против 7,8 % в сентябре. Эта платформа опередила Windows ХР, которая за месяц опустилась с третьей на четвёртую позицию: доля полюбившейся многим ОС сократилась с 9,1 % до 8,3 %.
В целом на различные версии Windows сейчас приходится 91,4 % рынка операционных систем для ПК. Ещё 6,4 % занимают платформы macOS. Доля Linux-систем равна 2,2 %.
