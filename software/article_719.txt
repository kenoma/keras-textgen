В Skype для Linux появились видеозвонки.
Microsoft постоянно выпускает обновления для переработанной Linux-версии Skype, релиз которой состоялся ранее в этом году. Но если до этого обновления включали в себя исправления багов и другие не слишком значимые изменения, то в Skype for Linux Alpha под версией 1.10 появилась функция, к добавлению которой разработчики стремились уже давно.
Как могли догадаться некоторые, речь идёт о видеозвонках. В анонсе Microsoft пишет: «На этой неделе у нас есть для вас нечто особенное. Команда изо всех сил трудилась над тем, чтобы добавить на Linux поддержку видеозвонков, и сегодня у нас готова для вас экспериментальная версия. Мы ещё не полностью готовы. Видеозвонки «один на один» работают пока только между клиентами Skype for Linux Alpha. Несмотря на раннюю фазу, мы хотели бы попросить вас, сообщество Linux, помочь нам с тестированием. Пожалуйста, дайте нам знать, как у вас работает видео».
Таким образом, теперь пользователи Linux могут общаться друг с другом по видеосвязи. Вероятно, ждать появления поддержки звонков между пользователями различных операционных систем придётся не слишком долго.
В анонсе также говорится о том, что некоторые старые версии Skype for Linux Alpha перестанут функционировать — пользователям придётся обновить свои клиенты до новых версий. Пользователи версий 1.1-1.6 получат сообщения с просьбой обновиться до последней версии. Загрузить её можно в качестве DEB- и RPM-пакетов.
