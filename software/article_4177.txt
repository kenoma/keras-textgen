Владельцы Xbox One получили доступ к Cortana.
Компания Microsoft выпустила для Xbox One превью-обновление, добавляющее поддержку голосового помощника Cortana. Изначально Cortana должен был появиться на консоли ещё в прошлом году, однако затем было принято решение добавить помощника в юбилейном апдейте. Cortana будет доступна через приборную панель Xbox One — взаимодействовать с ассистентом можно будет через Kinect или наушники. Таким образом, теперь игровую приставку можно будет запускать с помощью фразы «Эй, Кортана».
Интеграция с Cortana пока будет доступна только в США, Великобритании, Италии, Германии и Испании. С помощью новой функции можно будет организовывать группы, просматривать активность друзей и выполнять ряд базовых действий. Также Cortana сможет распознавать реальные имена и игровые теги друзей. Полная функциональность помощника станет доступна пользователем через некоторое время — пока им придётся ограничиваться лишь некоторыми возможностями.
Обновление также включает в себя ряд других функций, имеющих отношение к планам Microsoft сделать опыт использования Xbox Store и Windows Store более цельным. Так, появился новый интерфейс с игровыми коллекциями, который позволяет с лёгкостью просматривать готовые к установке игры. Вдобавок плиточный интерфейс теперь располагается вертикально, а не горизонтально. Также разработчики улучшили функцию шеринга в приложении Xbox для Windows 10 — пользователи теперь могут делиться скриншотами и видеороликами из игр прямо через Twitter. Что касается магазина приложений, то теперь можно фильтровать игры по распродажам и гораздо быстрее получать доступ к наборам, сезонным абонементам и дополнениям.
Установка последнего превью-обновления производится по желанию. Примерно через неделю новые функции станут доступны большему числу пользователей. Microsoft объединяет программы тестирования обновлений для Windows 10 и Xbox One, чтобы получать более обобщённые отзывы по приложению Xbox для Windows 10 и приборной панели.
