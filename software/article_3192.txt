Microsoft отказалась от поддержки Skype для Windows Phone 7.
Не так давно Microsoft представила технологию видеосообщений для своего приложения Skype под платформу Windows Phone 8. А теперь в официальном блоге компания сообщила, что не планирует выпускать обновления для Skype на Windows Phone 7, хотя приложение останется в магазине и будет доступно для загрузки и использования функций чата, видео и голосовых звонков.
Стоит отметить, что запуск Windows Phone 8 состоялся менее года назад — 29 октября 2012 года, а накануне Microsoft с партнёрами активно продвигали смартфоны вроде Nokia Lumia 900. Теперь эти флагманы годовой давности так безнадёжно устарели с точки зрения программной платформы, что Microsoft не видит смысла в поддержке ключевых сервисов на собственной ОС.
Сделано это, как объясняет Microsoft, с целью концентрации усилий на развитии версии Skype для Windows Phone 8, внедрения в неё новых возможностей и улучшения производительности. Компания обещает, что Skype для Windows Phone остаётся весьма важным продуктом и будет получать всё новые и новые функции.
Microsoft также напомнила, что за последний год было сделано многое для развития Skype. Приложение для Windows Phone 8 стала первой мобильной версией, которая получила поддержку HD-видеозвонков и новый пользовательский интерфейс. Skype становится одной из ключевых служб Microsoft: она доступна на мобильных системах, в настольном и сенсорном интерфейсе на ПК, а также интегрирована в Xbox One и Outlook.com.
