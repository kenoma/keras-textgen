Microsoft запустила бета-тест фреймворка глубокого обучения Cognitive Toolkit 2.0.
Microsoft анонсировала запуск бета-тестирования второй версии своего фреймворка глубокого обучения с открытым исходным кодом. Ранее программное обеспечение носило название Computational Network Toolkit (CNTK), однако теперь компания переименовала его в Cognitive Toolkit. Новое название более чётко соотносится с «когнитивной» маркетинговой стратегией Microsoft, используемой ею на протяжении последних нескольких месяцев.
Scott Eklund/Red Box Pictures
К Cognitive Toolkit теперь можно получить доступ не только через язык программирования C++: фреймворк получил поддержку Python 3, а позже будет добавлена и поддержка Python 2. Отсутствие поддержки этого высокоуровневого языка программирования было самым большим недостатком Cognitive Toolkit, заявил главный научный сотрудник в области исследования речи Сюэдон Хуан (Xuedong Huang) в интервью с VentureBeat. «Именно поэтому его внешняя адаптация была очень низкой», — заявил он.
Была улучшена производительность фреймворка, также была добавлена поддержка обучения с подкреплением — методики обучения машин путём проб и ошибок. И, наконец, теперь вместо того, чтобы запускать Cognitive Toolkit как скрипт, люди могут работать с программным обеспечением через интегрированную среду разработки Visual Studio.
Чем больше данных применяется к речевой акустической модели, тем более точной она становится.
Технология глубокого обучения используется многими продуктами Microsoft, включая Windows, Bing и Skype Translator. Cognitive Toolkit 2.0, по мнению редмондского гиганта, может помочь многим другим компаниям взять технологию на вооружение. К примеру, производитель холодильников Liebherr уже начал её использовать.
Cognitive Toolkit — программное обеспечение общего назначения, поэтому оно может работать со многими типами данных, включая речь, текст и изображения. Фаза тренировки легко может осуществляться на нескольких стандартных x86-процессорах (центральных и графических), что значительно ускоряет вычислительные процессы.
