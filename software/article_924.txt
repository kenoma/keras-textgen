Microsoft перестала продавать Windows 7 Professional производителям ПК.
Новые ПК под управлением операционных систем Windows 7 Professional и Windows 8.1 редко можно встретить в продаже, а скоро они совсем исчезнут, так как 31 октября Microsoft официально прекратила их OEM-поставки. Большинство производителей, правда, не дождались указанной даты, о которой было известно заранее, и давно перешли на Windows 10. Тем не менее, такие бренды, как, к примеру, Dell, до сих пор продолжали предлагать покупателям версии своих компьютеров с предустановленной Windows 7 Professional, учитывая тем самым интересы тех, кто так и не принял «десятку».
bhphotovideo.com
ПК на Windows 7 Professional и Windows 8.1 продолжат выпускаться до тех пор, пока у производителей остаётся некий запас ранее приобретённых у Microsoft лицензий. Что касается уже проданных ПК с данными ОС, то пока их владельцам беспокоиться не о чем. Windows 7 Professional, находясь на этапе расширенной поддержки, продолжит получать обновления безопасности вплоть до января 2020 года, а для Windows 8.1 и вовсе ещё действует программа основного обслуживания, которая завершится в 2018 году. Расширенная поддержка Windows 8.1 продлится до 2023 года.
blogs.windows.com
Но, несмотря на все усилия Microsoft, Windows 7 остаётся самой популярной в мире операционной системой для ПК. По данным Net Applications, в октябре 2016 года под её управлением работало 48 % всех компьютеров, что вдвое больше доли Windows 10. Кампания, в рамках которой все обладатели лицензионных копий Windows 7 SP1 и Windows 8.1 могли абсолютно бесплатно перейти на Windows 10, завершилась 29 июля нынешнего года.
