Экспериментальное приложение от Microsoft выводит копирование текста на новый уровень.
Microsoft Garage, экспериментальное подразделение редмондской компании, выпустило приложение для Android, которое позволяет легко и быстро копировать текст с экрана мобильного устройства. Оно анализирует «сырой» текст и даёт возможность с большой лёгкостью выбирать любой его участок для последующего копирования, отправки по почте или, например, добавления в список задач в каком-нибудь стороннем приложении.
Многие могут подумать, что новое приложение, получившее название Clip Layer, является очень примитивным и, по сути, представляет собой простой буфер обмена. Однако на самом деле не всё так просто: оно использует интерфейс программирования приложений Assist для поиска видимого текста и совершения с ним различных действий. Тот же API использует приложение Google — точнее, одна из его функций под названием Now on Tap, которая ищет на экране потенциально интересные элементы и показывает их в виде интерактивных карточек.
Clip Layer работает следующим образом: при удерживании домашней кнопки вокруг разных элементов текста образуются тонкие рамки. Нажатие на контент в одной из рамок его затемняет, после чего можно применить к выбранному тексту действия «копировать», «задача», «отправить по почте» или «поделиться». За действия эти отвечают кнопки в нижней части экрана.
Clip Layer работает не идеально: кое-какой текст оно не определяет. Тем не менее, приложение может быть полезно, поскольку зачастую копировать и вставлять текст на смартфоне бывает очень неудобно.
Одним из разработчиков Clip Layer является Стив Вон (Steve Won), старший дизайнер в команде Microsoft Office. Ранее он работал в Microsoft Garage над клавиатурой Hub Keyboard, в которой, помимо всего прочего, есть кнопка для быстрого копирования текста в буфер обмена.
