Виртуальный Форум Microsoft «Данные. Технологии. SQL Server 2016» состоится 8 июня.
8 июня состоится Виртуальный Форум Microsoft «Данные. Технологии. SQL Server 2016». Начало форума — в 10:00 мск. Принять участие в нём смогут все желающие. Для этого потребуется лишь пройти регистрацию.
Команда Microsoft подготовила для вас особенный, Виртуальный формат форума, который не отстает от продвинутого содержания докладов. Также были приглашены эксперты и партнёры для общения в интерактивном режиме.
Форум посвящён ключевыми технологиям SQL Server 2016, которые непременно войдут в арсенал администраторов и разработчиков баз данных и транзакционных приложений. Здесь можно будет обсудить любые аспекты современной бизнес-аналитики: корпоративной, самостоятельной, мобильной и предиктивной, также будет представлено новое поколение решений: IoT, Blockchain, Smart Personal Assistant, Боты, Cortanа Intelligent Suit.
В рамках форума запланированы три параллельных технологических сессии:
SQL Server 2016: новые стандарты в мире транзакции.
Бизнес-aналитика: SQL, Power BI, R, Mobile.
Azure: новое поколение решений для аналитики и Big Data.
С докладами на форуме выступят: Алексей Халяко, руководитель программ Customer Advisory Team (Microsoft, Германия), Олонцев Сергей, администратор и разработчик баз данных (Лаборатория Касперского, Россия), Павел Баркетов, технический директор компании «Софтпоинт» (Россия), Дмитрий Костылев, руководитель группы разработки баз данных (Лаборатория Касперского, Россия), Дмитрий Короткевич, MVP по платформе данных Microsoft, Алексей Князев, ведущий разработчик баз данных (Альфа-Банк, Россия) и другие.
Более подробно о программе Виртуального Форума Microsoft «Данные. Технологии. SQL Server 2016» можно можно узнать на сайте мероприятия.
