Todoist начал использовать ИИ для планирования задач.
В популярном приложении для составления списков личных и рабочих задач Todoist появилась функция планирования с помощью искусственного интеллекта. Теперь, когда пользователю нужно выбрать дату выполнения той или иной задачи, Todoist будет анализировать ряд различных факторов и выбирать для человека оптимальную дату. Эта возможность была добавлена в сервис с целью ограничить людей в выборе дат, поскольку некоторые из них зачастую ставят себе невыполнимые цели.
Мобильные и настольные приложения Todoist бесплатны для загрузки, но разработчики предлагают пользователям премиум-подписку за $29 в год, которая открывает доступ к функциям вроде напоминаний, пользовательских фильтров и прикрепляемых файлов. Новая функция, использующая возможности искусственного интеллекта и получившая название Smart Schedule, будет доступна бесплатно даже без подписки.
Smart Schedule будет адаптироваться под пользователя и анализировать такие вещи, как его прошлые привычки, запланированные рабочие задачи и типы задач, которым человек отдаёт предпочтение по рабочим и выходным дням. Функция также будет проводить крупномасштабный анализ данных для определения срочности задачи на основе информации, полученной ото всех пользователей Todoist. Таким образом, задача вроде чтения книги будет считаться менее приоритетной, чем назначение встречи.
Компания решила добавить в приложение такую функцию после того, как обнаружила особую группу пользователей, собирающих десятки задач каждую неделю, пока у них не накапливается по 50 задач на один день. «Когда у вас на день назначено больше 50 задач, ваш список дел становится куда менее эффективным инструментом для расстановки приоритетов», — заявил исследователь Олег Шидловский, возглавивший разработку Smart Schedule.
