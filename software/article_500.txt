Microsoft запустила сервис для создания ботов в облаке.
Microsoft запустила Azure Bot Service — сервис, позволяющий разработчикам переносить своих ботов в облако. По словам генерального директора Microsoft FUSE Labs Лили Ченг (Lili Cheng), сервис был создан для того, чтобы «дать вам операционную гибкость, чтобы работать и обрабатывать данные, пока ваши боты растут в популярности, причём платите вы только за то, чем пользуетесь». «Более того, вам также не нужно беспокоиться об обеспечении или управлении серверами, на которых работают ваши боты. Исправление и поддержание инфраструктуры осуществляется за вас — вы концентрируетесь на написании кода», — добавила Ченг. Ранее ни один из трёх крупнейших облачных провайдеров (Microsoft Azure, Amazon Web Services и Google Cloud Platform) не предоставлял возможности создавать ботов в облаке.
Напомним, что Microsoft Bot Framework позволяет создавать ботов, совместимых более чем с полудюжиной платформ, включая Skype, Microsoft Teams, Facebook Messenger и Kik. Боты, использующие Azure Bot Service, будут автоматически добавляться в каталог ботов компании. В прошлом месяце в интервью с VentureBeat Ченг рассказала о желании создать с помощью разработчиков и авторов крупнейших мессенджеров более надёжный и полный каталог ботов или даже движок для их поиска.
До одного миллиона серверных запросов Azure Bot Service бесплатен для использования. Сервис также предоставляет доступ к шаблонам разных ботов. В их список, например, входит бот, помогающий заполнять формы, а также бот, использующий интеллектуальную службу понимания языка (LUIS) — основную технологию обработки естественного языка, предоставляемую Microsoft Cognitive Services.
