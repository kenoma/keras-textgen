Windows 10 укрепляет позиции на мировом рынке ОС.
Компания Net Applications обнародовала июльскую статистику по мировому рынку операционных систем для персональных компьютеров.
Самой распространённой ПК-платформой в глобальном масштабе остаётся Windows 7, однако позиции этой ОС ослабевают. Так, если в июне её доля равнялась 49,1 %, то по итогам июля — 47,0 %.
Сокращение числа компьютеров с Windows 7 объясняется переходом пользователей на платформу Windows 10: обновление до этой операционной системы вплоть до 29 июля было бесплатным. В результате Windows 10 за прошедший месяц сумела укрепить позиции на 2 % и теперь удерживает 21,1 % глобального рынка.
На третьем месте по распространённости находится операционная система Windows ХР с результатом 10,3 %. Эта ОС опережает существенно более молодую Windows 8.1, доля которой по итогам июля составила 7,8 %.
В целом на операционные системы семейства Windows приходится немногим менее 90 % мирового рынка ПК-платформ. Ещё 7,9 % удерживают системы macOS. На Linux приходится около 2,3 %.
