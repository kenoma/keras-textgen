Новые функции Chrome для Android: экономия трафика и скачивание веб-страниц.
Приложение для офлайн-просмотра видео YouTube Go было не единственной новинкой в области софта для платформы Android, которую корпорация Google анонсировала накануне. Интернет-гигант также представил обновлённый браузер Chrome для этой ОС, финальный релиз которого состоится в октябре, а бета-версия доступна уже сейчас.
Оптимизация веб-страниц в обновлённом Chrome для Android
Одно из главных нововведений в Chrome понравится тем, у кого медленный или ограниченный по трафику Интернет. Функция называется Data Saver и уменьшает объём загружаемых данных на 60 %, сжимая не только текст и изображения, но и видео. Как утверждают разработчики, при воспроизведении MP4-видео из Сети браузер сокращает входящий трафик на величину до 67 %. Кроме того, при медленном соединении Data Saver оптимизирует веб-страницы, оставляя только базовые элементы их интерфейса, благодаря чему экономится до 90 % трафика, а скорость загрузки сайтов удваивается.
Скачивание веб-страниц, изображений и видео в обновлённом Chrome для Android
Ещё одна новая возможность Chrome для Android — скачивание веб-страниц, музыки, изображений и видео для последующего автономного доступа. В случае сбоя соединения закачки возобновляются автоматически. Для управления загрузками рядом в браузере появилось отдельное меню, с помощью которого можно открывать ранее скачанный контент, делиться им, просматривать историю и т. д.
