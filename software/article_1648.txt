Windows 10 занимает почти четверть мирового рынка операционных систем.
Компания Net Applications обнародовала статистику по мировому рынку операционных систем для персональных компьютеров по итогам сентября нынешнего года.
Самой распространённой программной платформой для ПК остаётся Windows 7, хотя доля этой ОС продолжает снижаться. Если год назад Windows 7 была установлена на 56,5 % персональных компьютеров с подключением к Интернету, то теперь — на 48,3 %.
На втором месте по популярности находится Windows 10. Данная платформа занимает почти четверть мирового рынка — 22,5 %, если быть точным. В сентябре 2015-го показатель составлял 6,6 %. То есть за год Windows 10 укрепила позиции практически в четыре раза.
Замыкает тройку Windows ХР с результатом 9,1 %. Годом ранее эта операционная система занимала приблизительно 12,2 % глобального рынка.
В целом, на различные версии Windows сейчас приходится 90,9 % отрасли. Ещё 6,9 % занимают операционные системы семейства macOS. Доля Linux равна 2,2 %.
