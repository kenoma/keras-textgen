Microsoft уверяет, что по энергоэффективности Edge недосягаем для конкурентов.
Microsoft в рамках рекламной кампании по продвижению своего браузера Edge при каждом удобном случае акцентирует внимание на пониженном расходе энергии, улучшениях JavaScript и долгожданной поддержке расширений, которая появилась в прошлом месяце. Недавно компания представила результаты очередной серии тестов, которые призваны продемонстрировать превосходство Edge над конкурентами по части энергоэффективности.
Microsoft уверяет, что обновлённый Edge «по-прежнему является лучшим выбором» для тех, кто хочет добиться максимальной продолжительности автономной работы на Windows 10. Уточняется, что последние улучшения повысили энергоэффективность Edge на 12 %, если сравнивать с версией, представленной в Windows 10 1511.
Разумеется, Microsoft не упустила возможность сопоставить энергоэффективность своего браузера с решениями конкурентов: Chrome, Firefox 47 и Opera (с функцией Battery Saver). Все тесты проводились на одинаковых устройствах с Windows 10 Anniversary Update. Результаты представлены ниже.
На графике видно, что эффективность Edge на 24–43 % оказалась выше при воспроизведении видео, во время активности в социальных сетях, а также при работе с несколькими активными вкладками. В будущем Microsoft обещает повысить эффективность использования циклов центрального процессора и минимизировать влияние фоновой активности браузера на производительность системы.
