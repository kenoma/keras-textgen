Windows 10 заняла почти четверть рынка операционных систем для ПК.
Компания Net Applications обнародовала статистику по мировому рынку операционных систем для персональных компьютеров по итогам августа 2016 года.
Windows 10 продолжает укреплять позиции: флагманская программная платформа Microsoft заняла почти четверть глобального рынка — 23,0 %, если быть точным. Для сравнения: годом ранее на долю этой ОС приходилось 5,2 %.
Тем не менее, пока Windows 10 существенно отстаёт от самой популярной операционной системы — Windows 7, доля которой равняется 47,3 %. Впрочем, эта ОС сдаёт позиции: так, в августе 2015-го она занимала 57,7 % мирового рынка.
На третьем месте с результатом 9,4 % располагается операционная система Windows XP, которая в этом году отметила своё пятнадцатилетие. Годом ранее рыночная доля этой платформы равнялась 12,1 %.
В целом на различные версии Windows сейчас приходится 90,5 % рынка операционных систем для ПК. Ещё приблизительно 7,4 % удерживает macOS. Доля различных вариантов Linux — около 2,1 %.
