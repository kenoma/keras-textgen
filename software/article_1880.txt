Microsoft работает над кроссплатформенным приложением Skype.
На данный момент существует множество вариантов Skype: версии для Windows, Windows RT, macOS, универсальное приложение для Windows 10, мобильные приложения для Android, iOS и Windows Phone и, конечно, веб-версия. Но Microsoft собирается избавиться от всего этого многообразия. По крайней мере, так утверждают источники, сообщившие изданию Ars Technica о том, что компания в настоящее время работает над единым кроссплатформенным приложением, носящим кодовое название Skype for Life.
Разработка Skype for Life ведётся в штаб-квартире Microsoft в Редмонде. По словам источников, создание клиента является сейчас приоритетной задачей для специалистов компании, а потому появление каких-либо новых функций для текущих версий при таких обстоятельствах маловероятно. Неизвестно, когда новинка может увидеть свет, и Microsoft пока никак не комментировала эту информацию.
К слову, именно с разработкой Skype for Life связывают закрытие офиса Skype в Лондоне и сокращение 220 сотрудников, работавших там, а также ещё 300, трудившихся в Редмонде. Сама Microsoft заявила, что такие меры связаны с планами по объединению ряда позиций.
