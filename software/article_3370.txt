Глава Microsoft: HoloLens — отличная платформа для Pokemon GO.
Pokemon GO успешно шагает по планете, а заодно стремительно повышает стоимость акций Nintendo. Даже исполнительный директор Microsoft Сатья Наделла (Satya Nadella) не смог не признать привлекательности этой игры. У Pokemon GO и Microsoft есть нечто общее: дополненная реальность. Поэтому неудивительно слышать от господина Наделлы, что Pokemon GO будто создана для очков HoloLens и подходит для них даже сильнее, чем для любого смартфона.
Действительно, в текущем воплощении (по крайней мере, без аксессуара Pokemon GO Plus) для ловли покемонов необходимо доставать смартфон и смотреть на окружающий мир через его камеру. Потенциальная версия игры для HoloLens позволила бы сделать процесс более естественным.
Разумеется, это вовсе не означает более интересного игрового окружения. Остаётся проблема средств ввода: пользователю либо придётся использовать какой-то дополнительный контроллер, либо прибегать к смущающим жестам на публике. Так или иначе, может быть, в будущем мы действительно увидим какую-нибудь версию Pokemon GO для HoloLens: корпорация Microsoft, похоже, эту идею поддерживает — дело только за разработчиками из Niantic Labs.
Любопытно, что в настоящее время Pokemon GO недоступна не только на платформе HoloLens, которая предлагается лишь в версии для разработчиков — AR-игра отсутствует на всех устройствах Windows, будь то мобильные ПК или, что куда важнее, смартфоны на базе Windows 10 Mobile.
