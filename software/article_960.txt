Деловые приложения Flow и PowerApps от Microsoft вышли из превью-стадии.
Microsoft анонсировала выход из превью-стадии двух своих сервисов — приложения для автоматизации задач Flow и инструмента для лёгкого создания бизнес-приложений PowerApps. Первоначальный запуск PowerApps состоялся в прошлом году, превью-версия его, как и превью-версия Flow, стала доступна в апреле. С тех пор обоими инструментами воспользовались уже более 160 тысяч людей из 71 тысячи организаций, заявил корпоративный вице-президент Microsoft Джеймс Филлипс (James Phillips).
Оба приложения теперь доступны каждому в App Store и Google Play. Более того, существует универсальное приложение PowerApps для Windows, а также веб-версия Flow. Сервисы входят в состав облачной службы Dynamics 365, а также доступны по подпискам Office 365 Enterprise, Business Premium и Essentials.
PowerApps
Помимо всего прочего, PowerApps теперь использует преимущества так называемой Common Data Model («модель общих данных»). «Common Data хранит ваши ключевые бизнес-данные в защищённой базе данных под управлением Microsoft Azure, организуя их в стандартизированную, но расширяемую форму», — написал Филлипс. «Стандартизация упрощает создание новых приложений и рабочих процессов, извлекающих пользу из ваших данных. И PowerApps, Flow и Power BI по умолчанию поддерживают эту модель общих данных, что сильно всё упрощает».
Flow
PowerApps — это современный способ создания приложений, позволяющий использовать данные из специальных корпоративных источников, включая SharePoint. В то время как PowerApps и Flow используют мощности центров обработки данных Microsoft Azure, созданные через PowerApps приложения могут задействовать данные локальных дата-центров различных компаний. Flow, в свою очередь, может использовать локальные «шлюзы» и в то же время привязываться к онлайн-сервисам.
