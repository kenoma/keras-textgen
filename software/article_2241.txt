Windows XP отмечает пятнадцатилетие.
Пятнадцать лет назад, 24 августа 2001-го, в производство отправилась операционная система Windows XP. Тогда ещё никто не предполагал, что эта платформа станет одним из самых успешных продуктов Microsoft.
В отличие от предыдущей системы Windows 2000, которая поставлялась как в серверном, так и в клиентском вариантах, Windows XP позиционировалась исключительно как клиентская платформа. Её серверным аналогом являлась Windows Server 2003.
Рядовым пользователям Windows XP стала доступна 25 октября 2001 года. Сразу после выхода операционная система начала стремительно набирать популярность. По данным Net Applications, Windows XP была самой распространённой операционной системой на компьютерах с доступом в Интернет вплоть до августа 2012 года: только тогда пальму первенства перехватила Windows 7 (к слову сказать, предшественнице данной ОС — Windows Vista — сделать этого не удалось).
Несмотря на то, что основная поддержка Windows ХР была остановлена ещё в 2009 году, а расширенная поддержка прекратилась в апреле 2014-го, эта платформа до сих пор пользуется весьма большой популярностью. Так, в конце июля нынешнего года Windows ХР удерживала 10,3 % мирового рынка ОС для персональных компьютеров. Это больше, чем у Windows 8.1 и Windows 8 вместе взятых, которые занимают 9,9 % отрасли.
Сейчас многие современные программы уже не работают в среде Windows XP, а для нового оборудования отсутствуют необходимые драйверы. Тем не менее, многие пользователи не спешат отказываться от полюбившейся платформы и переходить на более новые версии ОС. С пятнадцатилетием, Windows ХР!
