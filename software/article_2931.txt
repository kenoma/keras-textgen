Android N — главные нововведения мобильной ОС.
Google в рамках конференции для разработчиков Google I/O официально представила новую мобильную операционную систему Android N. На презентации разработчики продемонстрировали основные нововведения мобильной ОС, попутно показав голосового ассистента, мессенджер, вспомогательную систему для дома и многое другое. Первые две сборки мобильной операционной системы уже доступны для разработчиков. Но Google во время презентации не забыла рассказать, чем Android N будет полезна и для простых пользователей. Кстати, официальная расшифровка таинственной буквы N на этот раз представлена не была. Разработчики предложили всем желающим самим придумать название для системы.
SafetyNet
Очень много внимания при разработке новой мобильной ОС уделялось повышению производительности и безопасности использования. Так, Android N снабдили специальной системой защиты SafetyNet. Она будет отвечать за проверку устанавливаемых программ вне зависимости от их источников.
Всем тем, кто привык пользоваться взломанными версиями популярных платных программ, придётся несладко. SafetyNet будет сама удалять любые подозрительные версии приложений, если содержимое их дистрибутива будет отличаться от образа из Google Play.
Многооконный режим
Следующее важное нововведение Android N — появление полноценного многооконного режима. Приятно, что данная функция будет работать практически со всеми выпущенными программами, она не требует внесения каких-либо серьёзных видоизменений в исходном коде.
Быстро переключаться между двумя приложениями разработчики предлагают с помощью специальной кнопки многозадачности. В Android N появилась возможность закрытия запущенных приложений с помощью одной кнопки, дополнительно система сможет показать приложения, которыми давно не пользовались. Также был добавлен режим «картинка в картинке».
Панель уведомлений подверглась полной перерисовке. Теперь, например, появилась возможность отвечать на уведомления от социальных сетей и мессенджеров прямиком из этой самой шторки.
Поддержка API Vulkan
Важное нововведение для всех любителей мобильного гейминга. Android N полностью поддерживает Vulkan.
Как утверждают представители Google, внедрение данного кроссплатформенного API для 2D и 3D-графики позволит разработчикам игр эффективнее использовать ресурсы устройства, что должно положительно сказаться на уровне выдаваемой графики и общей производительности приложения.
Режим энергосбережения Doze
Данная функция появилась на Android ещё в прошлом году. В Android N она будет по умолчанию активироваться всякий раз, когда экран устройства будет выключаться.
Вместе с этим в Android N проведена серьёзная работа по оптимизации работы гаджетов, оснащённых оперативной памятью небольшого объёма. Управление работой приложений в фоновом режиме тоже было доработано.
Экономия трафика
Data Saver — новый режим Android N, позволяющий экономить потребление интернет-трафика мобильным устройством. В таком режиме потребление данных приложений в фоновом режиме останавливается.
У пользователя появится возможность самому выбирать, для каких программ ограничения Data Saver применяться не будут.
Google Assistant
Это логичное продолжение идей, заложенных в Google Now. Виртуальный помощник будет ещё больше походить на Siri для iOS. Работа с программой происходит в режиме диалога. Важно, что Google Assistant учитывает при анализе вопросов не только его основное содержание, что само собой разумеется, но ещё внешние условия (месторасположение, например) и контекст.
Платформа виртуальной реальности Daydream
Во время презентации Android N компания Google представила новый режим интерфейса системы, специально «заточенный» под использование в виртуальной реальности.
Уже сейчас над играми и приложениями для Daydream работает множество крупных компаний, среди которых есть Netflix, HBO и CNN. На презентации даже показали новый прототип VR-гарнитуры, который по своей сути является улучшенной версией известного Google Cardboard.
Официальный релиз новой Android N должен состояться до конца этого лета. Предложения по расшифровке той самой буквы N в названии можно отправлять по данному адресу.
