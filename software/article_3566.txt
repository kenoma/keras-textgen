Windows 10 занимает пятую часть рынка операционных систем для ПК.
Компания Net Applications обнародовала статистику по распространению операционных систем для персональных компьютеров в мировом масштабе по итогам июня нынешнего года.
Самой распространённой программной платформой для ПК остаётся Windows 7, занимающая приблизительно 49,1 % глобального рынка. Однако позиции этой ОС слабеют: так, год назад она применялась на 61,0 % персональных компьютеров.
Причина снижения доли Windows 7 — выход операционной системы Windows 10 в конце июля 2015-го. Сейчас новейшая платформа Microsoft удерживает практически пятую часть рынка — 19,1 %, если быть точным.
Замыкает тройку самых распространённых ОС платформа-ветеран Windows ХР с результатом в 9,8 %. Для сравнения: годом ранее эта операционная система занимала около 12 % отрасли.
В целом на различные версии Windows приходится 89,8 % глобального рынка. Ещё 8,2 % занимают платформы OS X компании Apple. На операционные системы Linux в сегменте персональных компьютеров приходится только около 2 %.
