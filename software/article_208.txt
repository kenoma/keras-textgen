Обновление Redstone 3 изменит внешний вид Windows 10.
Разрабатывая Windows 8, корпорация Microsoft отказалась от графического интерфейса Aero, использовавшегося в Windows 7, что вызвало негодование части пользователей, которым нравилось, как выглядела на их ПК «семёрка». Тем не менее, редмондский софтверный гигант продолжил развивать тему Metro UI и в последующих своих продуктах, в результате чего мы получили внешний вид Windows 10.
Впрочем, как сообщает интернет-издание WindowsCentral со ссылкой на неназванные источники, Microsoft уже больше года работает над «рестайлингом» Windows 10. Проект носит кодовое название NEON и, предположительно, станет частью обновления Redstone 3, которое выйдет осенью 2017 года, а участникам программы Windows Insider станет доступно на несколько месяцев раньше — в апреле–мае.
Подробностей о Project NEON пока немного: сообщается лишь то, что переработанный UI станет плавнее нынешнего, в нём появятся новые анимации и переходы. Одним из самых заметных нововведений обещает стать возможность выносить элементы приложений за пределы их окон. Ожидается, что NEON объединит принципы классического рабочего стола с дополненной реальностью и голографическим окружением, для чего будут применяться текстуры, 3D-модели и приёмы с освещением.
Как это будет выглядеть, пока можно только догадываться, но революция с точки зрения оформления Windows 10 всё же вряд ли грядёт, ведь, согласно имеющейся информации, Project NEON основан на текущей концепции MDL2, а не заменяет её. Зато его разработчики обещают навести порядок с оформлением приложений, выпустив рекомендации для создателей софта по приведению дизайна их продукции к единому «знаменателю».
