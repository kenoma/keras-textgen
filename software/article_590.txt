Microsoft анонсировала запуск Visual Studio на macOS.
Microsoft объявила о том, что скоро её серия продуктов для программирования Visual Studio станет доступна владельцам компьютеров на базе macOS. Многие профессиональные пользователи ждали этого уже довольно давно. Для редмондской компании такое решение не совсем обычно, поскольку раньше она всеми силами старалась удерживать разработчиков в рамках своей платформы, выпуская инструменты для программирования исключительно на Windows.
Таким образом, компания начинает ещё больше ориентироваться на облачные вычисления. Если в прошлом многие задачи по программированию осуществлялись на локальных серверах, то теперь разработчики всё больше переключаются на облачные сервисы вроде Azure от Microsoft и AWS от Amazon. Делая Visual Studio мультиплатформенным продуктом, компания надеется удержать как можно больше программистов, позволив им работать над своими продуктами с какой угодно операционной системы. Visual Studio для macOS базируется на платформе для разработки приложений Xamarin Studio, и данное изменение может вдохновить авторов программ для macOS и iOS на создание приложений для Windows, поскольку у них больше не будет нужды обзаводиться отдельным компьютером на Windows или устанавливать виртуальную машину для доступа к Visual Studio.
Ранее в этом году Microsoft объявила о добавлении в Windows поддержки командной оболочки Bash, позволяющее запускать на ОС от редмондской компании окружение Ubuntu. Тогда специалист по веб-технологиям Microsoft Скотт Хансельман (Scott Hanselman) заявил, что это «блестящий вариант для разработчиков, использующих разнообразный набор инструментов». Запуск Visual Studio для macOS — очень похожий ход.
Превью-версия программного обеспечения будет представлена на конференции Microsoft Connect(), которая пройдёт 16–18 ноября.
