Bing для Android оснастили функцией поиска по изображению и сканером штрих-кодов.
Microsoft выпустила очередное обновление для приложения Bing Search на Android. Свежий патч добавляет в мобильную версию поисковика несколько важных функций, которые делают работу с приложением более комфортной и быстрой.
Во-первых, Bing для Android поддерживает поиск по изображению. Теперь пользователю достаточно прикрепить нужную картинку и задать запрос. Полноценная версия поисковика Google, например, поддерживает данную функцию уже в течение нескольких лет, но в Bing на Android она появляется только сейчас.
Во-вторых, мобильный поисковик от Microsoft теперь «обучен» давать пользователю новые предложения по поиску изображений и видео всякий раз, когда он решит выделить ключевое слово в тексте. Наконец, Bing для Android оснастили встроенным сканером штрих-кода. По мнению Microsoft, реализация данной функции поможет владельцу Android-устройства упростить шопинг, прибегая к помощи крупнейших онлайн-площадок, наподобие eBay, Best Buy и других.
Последняя версия Microsoft Bing уже доступна в Play Store. Разработчики приложения очень гордятся популярностью поисковика на конкурирующей платформе, отмечая небывалый рост количества загрузок.
