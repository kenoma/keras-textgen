Виртуальный форум Microsoft «Данные. Технологии. SQL».
Открыта предварительная регистрация участников Виртуального Форума Microsoft «Данные. Технологии. SQL Server 2016», который пройдёт 8 июня 2016 года. Принять участие в нём смогут все желающие, причём бесплатно. Начало форума — в 10:00 мск.
Мероприятие приурочено к выходу новой версии платформы управления данными Microsoft на базе SQL Server 2016 — лидера среди систем управления базами данных.  На форуме вы познакомитесь с ключевыми технологиями SQL Server 2016, которые непременно войдут в арсенал администраторов и разработчиков баз данных и транзакционных приложений. Также будут обсуждаться все аспекты современной бизнес-аналитики: корпоративной, самостоятельной, мобильной и предиктивной. Кроме того, будет представлено новое поколение решений: IoT, Blockchain, Smart Personal Assistant, Боты, Cortanа Intelligent Suit.
В рамках форума запланированы три параллельных технологических сессии:
SQL Server 2016: новые стандарты в мире транзакции.
Трек посвящён ключевым технологиям    SQL  Server     2016, которые непременно войдут в арсенал администраторов и разработчиков баз данных и транзакционных приложений. Доклады освещают такие аспекты, как создание катастрофоустойчивых конфигураций, управление безопасностью, увеличение производительности с помощью кластеризации или переноса данных в оперативную память. Другая важная тема трека — инструменты    SQL  Server     2016, которые упрощают выполнение ежедневных задач администрирования и разработки.
Бизнес-aналитика: SQL, Power BI, R, Mobile.
В этом треке вы узнаете всё о современной бизнес-аналитике на платформе SQL 2016. Вам расскажут о сервисе для самостоятельной аналитики Power BI, будут рассмотрены возможности мобильной аналитики, а также состоится обсуждение Прогнозной аналитики (АА) и системы визуализации. Помимо этого, будут подробно освещены самые ожидаемые новинки SQL2016 — встроенная аналитика на базе R в SQL Server 2016, и компонент Polybase, позволяющий анализировать данные в Hadoop с помощью традиционного языка запросов T-SQL. Современная бизнес-аналитика на платформе SQL 2016 открывает новые возможности для развития бизнеса, ускоряет процесс принятия решений и повышает производительность.
Azure: новое поколение решений для аналитики и Big Data.
Мы уже вступили в эру, когда необходимы новые инструменты для решения современных проблем, которые требуют работы с массивами данных. Важно владеть современными подходами к работе с технологиями, чтобы отвечать новым технологическим задачам. В докладах будут рассмотрены основные сервисы для реализации амбициозных планов по работе с данными. Мы обсудим новое поколение решений IoT и Blockchain, Smart personal assistant и инструменты: Bot framework, Maсhine Learning и другие.
С подробностями программы Виртуального Форума Microsoft «Данные. Технологии. SQL Server 2016» можно ознакомиться на сайте мероприятия.
