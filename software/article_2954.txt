«Юбилейное обновление» для Windows 10 Mobile выйдет 9 августа.
«Юбилейное обновление» для Windows 10 уже стало доступно владельцам компьютеров и планшетов на базе операционной системы Microsoft, теперь же настала очередь смартфонов. Как заявил редмондский гигант, обновление для Windows 10 Mobile станет доступно для загрузки 9 августа.
Мобильная версия Windows 10 получит примерно те же возможности, что и настольный вариант операционной системы, однако некоторые функции останутся эксклюзивны для смартфонов. Приложение Windows Camera обзаведётся панорамным режимом, а также экран блокировки будет усовершенствован таким образом, что с помощью сенсорной кнопки «назад» можно будет делать быстрые снимки на устройствах без соответствующих аппаратных кнопок.
Windows 10 Mobile с юбилейным обновлением также получит новое приложение Skype, аналогичное универсальному приложению для настольной версии ОС. Таким образом, владельцы смартфонов на базе операционной системы Microsoft наконец получат нормальную замену крайне неудобному приложению Skype для Windows Phone, которым пользователи вынуждены были довольствоваться последние несколько лет. В другие универсальные приложения также будет внесена масса изменений.
В приложении настроек будут обновлены иконки, благодаря чему сильно упростится поиск некоторых опций. Центр уведомлений можно будет настроить таким образом, чтобы сначала отображались самые важные сообщения. Также Microsoft добавит в Windows 10 Mobile кнопки управления медиа на экране блокировки, поддержку сканеров отпечатков пальцев и новое приложение Microsoft Wallet, позволяющее осуществлять бесконтактные платежи на смартфонах Lumia 650, 950, и 950 XL. Голосовой помощник Cortana получил функцию поиска смартфона. Наконец, список эмодзи пополнился большим числом новых смайлов.
