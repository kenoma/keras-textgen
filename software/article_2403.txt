AdDuplex: количество смартфонов с Windows 10 Mobile неуклонно растёт.
Сервис AdDuplex, обеспечивающий реализацию контекстной рекламы в приложениях для мобильной платформы Windows, также делится статистикой о популярности всех её версий и распространённости тех или иных моделей устройств под её управлением.
windows-news.com
К примеру, операционная система Windows 10 Mobile к текущему моменту времени установлена на 14 % всех активных смартфонов под управлением Windows. Аналогичный показатель в прошлом месяце составлял лишь 11,9 %, что свидетельствует о тенденции постепенного перехода на самую актуальную версию мобильной ОС разработки Microsoft.
Что касается ОС Windows Phone 8.1, то её доля составляет 77,3 %. Оставшиеся 6,7 % и 1,9 % поделены между Windows Phone 8 и Windows Phone 7.x соответственно.
Если более детально взяться за рассмотрение полученных апдейтов для Windows Mobile 10, то по данным AdDuplex, свыше 52,4 % смартфонов получили «Юбилейное обновление» и уже используют сборку 1607. Учитывая тот факт, что возможность перехода официально взяла свой старт в начале августа, а для операторских смартфонов Windows Mobile 10 build 1607 и вовсе стала доступной только на прошлой неделе (для аппаратов AT&T и вовсе текущей), в целом статистика для Microsoft кажется положительной.
Оставшиеся 41,3 % смартфонов работают на Windows Mobile 10 build 1511, а 6,3 % довольствуются тестовой сборкой Windows 10 Mobile для инсайдеров.
Самым же популярным аппаратом среди Windows-смартфонов на сегодня является модель Microsoft Lumia 535, которой принадлежит 12,3 % обозначенного сегмента рынка. Второе место занимает смартфон Nokia Lumia 520 с показателем 10,4 %, а третье — Microsoft Lumia 640 с его 9,8 %. На смартфоны под управлением Windows от сторонних производителей в общей сложности приходится 33,1 % всех активных устройств.
