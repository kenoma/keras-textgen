Microsoft выплатила $10 000 за принудительное обновление до Windows 10.
Несмотря на все преимущества ОС Windows 10, не все пользователи стремятся обновить программное обеспечение своего ПК, и агрессивная кампания по модернизации операционной системы Microsoft вызывает критику и протесты.

По крайней мере один пользователь получит компенсацию за невольное обновление операционной системы своего компьютера на Windows 7 до Windows 10. По данным Seattle Times, предприниматель из Калифорнии Тери Голдштейн (Teri Goldstein) подала в прошлом месяце в суд на Microsoft после того, как без уведомления была предпринята неудачная попытка произвести автоматическое обновление операционной системы её компьютера до Windows 10. В результате компьютер, использовавшийся Тери для нужд своего туристического агентства в Саусалито (Калифорния), стал часто зависать и работать нестабильно.
«Я никогда не слышала о Windows 10, — заявила в суде Тери Голдштейн. — Никто не спросил меня, хочу ли я произвести обновление».
После безуспешной попытки решить проблему при помощи техподдержки Microsoft, Тери подала в суд на компанию, потребовав возместить потери дохода из-за неисправного компьютера и выплатить сумму для покупки нового. В итоге суд постановил удовлетворить её иск и выплатить компенсацию в размере $10 000. Microsoft отказалась от апелляции во избежание дальнейших судебных издержек.
Это дело является лишь одним из длинного ряда жалоб на Microsoft, которая ведёт чрезмерно агрессивную кампанию по распространению Windows 10. Операционная система, предлагаемая до сих пор в качестве бесплатного обновления с Windows 7 или более поздней версии вплоть до 29 июля, поначалу позиционировалась в качестве опционального обновления, к которому пользователи должны были проявить интерес.
Затем Microsoft перевела Windows 10 в разряд «рекомендуемых обновлений» для Windows 7 или более поздней версии, а это означало старт её автоматической загрузки через Центр обновления Windows в феврале этого года.
В марте начали поступать жалобы пользователей на автоматическую установку Windows 10 на их компьютеры без разрешения, что приводило к различным проблемам в работе.
Принудительное обновление вызвало протесты пользователей, но, вместе с тем, улучшило показатели Windows 10. Согласно данным Netmarketshare, в конце марта ею пользовались на 270 млн компьютеров, что составляло 17,43 % общего количества компьютеров во всём мире, благодаря чему Windows 10 теперь уступает по популярности лишь Windows 7.
