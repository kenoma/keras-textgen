Windows 10 впервые заняла более четверти рынка ОС Microsoft для ПК.
Компания Net Applications опубликовала статистику по распределению различных версий операционных систем для персональных компьютеров по состоянию на конец ноября уходящего года.
Microsoft
Самой распространённой программной платформой на ПК остаётся Windows 7 с долей в 47,2 %. В октябре эта операционная система занимала 48,4 %, а в ноябре 2015-го — 56,1 %.
На втором месте находится Windows 10. Как отмечается, впервые за последние месяцы данная платформа заметно укрепила позиции: её доля достигла 23,7 % против 22,6 % месяцем ранее. Год назад ОС удерживала только 9,0 % рынка. Отмечается также, что в сегменте Windows-систем для ПК доля Windows 10 впервые перевалила за 25 %, достигнув 26,1 %.
Замыкает тройку самых популярных ОС платформа-ветеран Windows ХР с показателем 8,6 %. За год позиции этой операционной системы ослабли на 2 %.
В целом, на различные варианты Windows сейчас приходится 91 % рынка программных платформ для персональных компьютеров. Ещё 6,7 % занимает macOS; примерно 2,3 % приходится на Linux.
