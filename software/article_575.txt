Skype для Windows получил встроенную функцию перевода устной речи на другие языки.
Анонсированное Microsoft приложение Skype Translator прежде было доступно в качестве отдельной программы, позволявшей в режиме реального времени осуществлять перевод вашей устной речи на семь различных языков. Соответственно, пока алгоритм Skype Translator пребывал в режиме тестирования, в головном приложении данная функция по умолчанию отсутствовала.
Однако теперь Microsoft объявила, что выявленные недоработки Skype Translator после испытаний были устранены, а само приложение функционирует должным образом, поэтому новая версия программы Skype для Windows обзаведётся интегрированной опцией для перевода разговора на другие языки.

К слову, на данный момент самым популярным направлением перевода в Skype Translator является «французский–английский», а всего доступно семь языков: северокитайский, английский, французский, немецкий, итальянский, испанский и португальский.
Кроме того, Skype поддерживает перевод печатного текста на значительно большее количество языков, нежели заявлено для голосового перевода через Skype Translator. В дальнейшем мультиязычность обозначенной функции будет расширяться, а само приложение со встроенным в неё дополнением станет доступным не только для компьютеров с Windows, но и появится на других платформах.
